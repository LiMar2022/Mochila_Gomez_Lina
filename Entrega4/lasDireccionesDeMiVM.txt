3. ¿Es diferente a la IP pública y privada de nuestra máquina host?

  Sí, porque la dirección IP pública de una máquina virtual es la dirección que se utiliza para comunicarse con Internet, mientras que la dirección IP privada se utiliza para la comunicación dentro de la red local.
