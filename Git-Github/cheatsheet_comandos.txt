Comandos git

- Establecer el usuario y el e-mail
  git config --global user.name "nombre de usuario"
  git config --global user.email email@email.com


- Configurar la rama main por defecto
  git config --global init.defaultBranch main


- Crear un nuevo repositorio
  git init


- Verificar el estado de los archivos/directorios
  git status


- Añadir un archivo
  git add nombre_archivo_directorio 
  git add . / git add --all (todos los archivos)


- Historial de actividad
  git log --oneline


- Commitear un archivo/directorio
  git commit nombre_archivo -m "mensaje del commit"


- Remover un archivo
  git rm archivo
  git rm -r directorio (remueve el directorio y los archivos que contiene)


- Vincular el repositorio archivo local existente con uno remoto
  git remote add origin "nombre del repositorio remoto"


- Verificar que están vinculados los repositorios
  git remote -v


- Vincular un repositorio remoto a nuestro repositorio local
  git clone "nombre del repositorio remoto"


- Subir cambios al repositorio remoto
  git push origin main


- Bajar cambios del repositorio remoto
  git pull origin main


- Crear una rama
  git branch nombre_nueva_rama


- Moverse a otra rama
  git checkout nombre_nueva_rama


- Lista de las ramas
  git branch


- Comparar dos ramas
  git diff nombre_rama_1 nombre_rama_2
